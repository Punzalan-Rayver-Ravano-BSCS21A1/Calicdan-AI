Powershell Commands:

First step: go to the directory of your project folder
Command Example - cd Calicdan-AI

2nd step: Activate Venv
Command - ./venv/Scripts/activate

3rd step: go to the backend folder using the command (cd backend) and then Install the required packages for the API to work 
Command - pip install -r requirements.txt

4th step: set the API Key
Command - $env:OPENAI_API_KEY="sk-c8e10ae8c2f1420ea5deb7e74b68a433" python main.py

5th step:

Last Step: Activate or run the server
Command - uvicorn main:app --reload
